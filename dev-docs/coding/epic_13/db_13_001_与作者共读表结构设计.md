# Epic 13: 与作者共读 - 数据库表结构设计

根据 PRD《Epic 13: 与作者共读》的需求及最新的精简修改意见，设计以下相关数据库表结构及对现有表的调整。

## 1. 新增表：`books` (图书表)

用途：存储用户导入的各类书籍信息、解析状态及其阅读进度。

| 字段名 | 数据类型 | 约束/默认值 | 描述 |
| ------ | -------- | ----------- | ---- |
| `id` | Integer | Primary Key | 主键 |
| `title` | String(255) | Nullable=False | 书名（解析或用户定义） |
| `author` | String(128) | Nullable=True | 书籍原作者（区别于绑定的 AI 好友） |
| `cover_url` | String(1024) | Nullable=True | 书籍封面本地相对或绝对路径 |
| `file_name` | String(255) | Nullable=False | 原始文件名 |
| `file_path` | String(1024)| Nullable=False | 书籍在本地系统的绝对或相对存储路径（含后缀名表明文件格式） |
| `status` | String(32) | Default='imported'| 当前状态：`imported`, `processing`, `ready`, `limited`, `failed` |
| `status_detail` | Text | Nullable=True | 解析失败原因、受限提示或其它处理细节 |
| `ai_friend_id` | Integer | ForeignKey("friends.id"), Nullable=True | 绑定的 AI 好友 ID（伴读伴侣） |
| `reading_location`| String(255) | Nullable=True | 阅读进度定位值，用于记录用户退出时的位置（例如页码或 CFI 字符串），帮助用户下次恢复定位 |
| `create_time` | UTCDateTime | Default=now | 导入记录创建时间 |
| `update_time` | UTCDateTime | OnUpdate=now | 最后更新时间（包括阅读进度的频繁更新） |
| `deleted` | Boolean | Default=False | 软删除标记 |


## 2. 修改现有表：`chat_sessions` (会话表)

用途：为了满足“书籍维度会话隔离”的要求，将伴读聊天与单聊进行隔离。同一个 AI 好友绑定多本书或知识库时，每个来源必须拥有独立的伴读会话上下文。不能直接复用普通单聊或其他知识库的聊天历史。

**新增字段**：

| 字段名 | 数据类型 | 约束/默认值 | 描述 |
| ------ | -------- | ----------- | ---- |
| `session_type` | String(32) | Default='normal' | 会话类型：`normal` (普通单聊), `book_reading` (伴读聊天) 等 |
| `knowledge_id` | Integer | Nullable=True | 通用知识 ID，根据 `session_type` 指向不同的表（如 `books.id`），不设外键约束以保持扩展性 |

**说明**：通过 `session_type` 和通用化命名的 `knowledge_id` 可以独立出该书籍/知识库专属的伴读会话，避免跨书籍串台，且方便未来扩展更多类似文档对话、通用知识库对话的场景。`knowledge_id` 不绑定外键，由应用层根据 `session_type` 查询对应的表。

## 3. SQLAlchemy 模型实现参考代码 (Python)

下面是用作实施的 `server/app/models/book.py` 的精简基础模型定义。

```python
from sqlalchemy import Column, Integer, String, Text, Boolean, ForeignKey
from sqlalchemy.orm import relationship
from app.db.base import Base
from app.db.types import UTCDateTime, utc_now

class Book(Base):
    __tablename__ = "books"

    id = Column(Integer, primary_key=True, index=True)
    title = Column(String(255), nullable=False)
    author = Column(String(128), nullable=True)
    cover_url = Column(String(1024), nullable=True)
    file_name = Column(String(255), nullable=False)
    file_path = Column(String(1024), nullable=False)
    
    # imported, processing, ready, limited, failed
    status = Column(String(32), default="imported", nullable=False) 
    status_detail = Column(Text, nullable=True)
    
    # 绑定的伴读好友
    ai_friend_id = Column(Integer, ForeignKey("friends.id"), nullable=True)
    
    # 极简的阅读进度记录，帮助用户下次定位到之前的进度位置
    reading_location = Column(String(255), nullable=True)
    
    create_time = Column(UTCDateTime, default=utc_now, nullable=False)
    update_time = Column(UTCDateTime, default=utc_now, onupdate=utc_now, nullable=False)
    deleted = Column(Boolean, default=False, nullable=False)

    # 关联关系
    friend = relationship("Friend")
```
